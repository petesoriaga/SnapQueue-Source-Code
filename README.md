# SnapQueue Source Code Export

This repository contains bundled source-code documents for the SnapQueue web application.

Included files:

- `SnapQueue-Complete-Source-Code.html`
- `SnapQueue-Complete-Source-Code.rtf`

The HTML version includes:

- a table of contents
- per-file title sections
- search
- a back-to-top button

The RTF version is included for Microsoft Word compatibility.
